# Lior Project

Это мой проект, содержащий различные примеры и утилиты.

## Структура проекта

- **FASTAPI__** — примеры с FastAPI
- **flask** — примеры с Flask
- **HTML AND CSS** — примеры HTML и CSS
- **Мои штучки** — личные скрипты и утилиты

## Начало работы

### Требования
- Python 3.x
- Виртуальное окружение (`.venv`)

### Установка
```bash
# Активация виртуального окружения
.venv\Scripts\activate

# Установка зависимостей (при наличии requirements.txt)
pip install -r requirements.txt
```

## Запуск

Запустите нужный файл из соответствующей директории.
